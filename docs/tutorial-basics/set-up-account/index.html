Set up Account

In order to buy or sell on NumerBay, a NumerBay account with Numerai API key needs to be set up.

Sign up​

Head to numerbay.ai, click the account icon on the top right of the page, and:

Login via MetaMask​

Login via Metamask if you prefer easier authentication

Username sign up​

Sign up manually by clicking the Username Login tab, and click Register today to switch to the sign up form

Set up Numerai API Key​

After signing up and logging in, head to the NumerBay account page for the Numerai API tab under the Account sidebar section.

If you don't have a Numerai API key yet, you can create API Keys in the Numerai Account Settings page. Make sure it has at least View user info permission. NumerBay only uses user info for model ownership verification and email notifications.

Additional API permissions are needed depending on what you want to do on NumerBay, you can always change API keys later:
